Start Android Emulator from Terminal

Follow These Steps:

  • Step 1: Ensure Android SDK is Installed

    Install Android Studio or ensure the standalone Android SDK is installed. Add the SDK tools to your system's PATH:

    export PATH=$PATH:/path-to-android-sdk/tools:/path-to-android-sdk/platform-tools
  • Step 2: List Available Emulators

    Use the following command to list available Android Virtual Devices (AVDs):

    emulator -list-avds
  • Step 3: Start the Emulator

    Start the emulator using the name of your AVD:

    emulator @<AVD_NAME>

    Replace <AVD_NAME> with your AVD's name, e.g., Pixel_4_API_30.

  • Step 4: Optional Customizations

    Use additional options to customize the emulator startup:

    • Start with a cold boot: emulator @<AVD_NAME> -no-snapshot-load
    • Set the scale of the emulator: emulator @<AVD_NAME> -scale 0.5
  • Step 5: Verify Emulator is Running

    Check if the emulator is running using:

    adb devices

    You should see a device like emulator-5554 in the list.

Guide: How to Start an Android Emulator from Terminal

This guide walks you through the steps to start an Android Emulator using the terminal. Follow the instructions carefully to set up and run your emulator.

Step 1: Ensure Android SDK Is Installed

Before starting, make sure you have Android Studio installed and the Android SDK is set up properly.

  • Open Android Studio.
  • Go to File > Settings > Appearance & Behavior > System Settings > Android SDK.
  • Ensure Android SDK Tools and Android Emulator are installed.

Step 2: Add SDK Tools to PATH

To use the emulator command, you need to add SDK tools to your system's PATH.

export PATH=$PATH:/path/to/Android/sdk/platform-tools:/path/to/Android/sdk/emulator

Replace /path/to/Android/sdk with the actual location of your SDK.

Step 3: List Available Emulators

Run the following command to see a list of available emulators:

emulator -list-avds

This will display emulator names, e.g., Pixel_5_API_30.

Step 4: Start an Emulator

Use the following command to start an emulator:

emulator -avd AVD_NAME

Example:

emulator -avd Pixel_5_API_30

Step 5: Optional Parameters

Customize the emulator with optional parameters:

  • Scale: -scale 0.5
  • RAM: -memory 2048

Example:

emulator -avd Pixel_5_API_30 -scale 0.5 -memory 2048

Troubleshooting

If you encounter issues, try the following:

  • Kill stuck emulator processes: 
    adb kill-server
  • Clear Emulator Cache: Navigate to .android/avd in your home directory and delete temporary files.

Note: Ensure that your system meets the requirements for running Android emulators.
